# Altair has been updated! - v4.3.0

What's Changed

  • added support for providing multiple windows when initializing altair by @imolorhe in https://github.com/altair-graphql/altair/pull/1759
  • updated electron-updater by @imolorhe in https://github.com/altair-graphql/altair/pull/1760
  • Add setting to disable the update available notification (#1) by @fabianfrangella in https://github.com/altair-graphql/altair/pull/1757
  • Bump trim-newlines from 3.0.0 to 3.0.1 by @dependabot in https://github.com/altair-graphql/altair/pull/1768
  • Bump aws-sdk from 2.787.0 to 2.1030.0 in /packages/altair-app by @dependabot in https://github.com/altair-graphql/altair/pull/1750
  • use transformed data environment in subscription connection params hydration by @imolorhe in https://github.com/altair-graphql/altair/pull/1767
  • added new logic for generating queries and fragments. by @imolorhe in https://github.com/altair-graphql/altair/pull/1773
  • added query subcollections logic. by @imolorhe in https://github.com/altair-graphql/altair/pull/1775

New Contributors

  • @fabianfrangella made their first contribution in https://github.com/altair-graphql/altair/pull/1757
